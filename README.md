# Assignment 10 – Pandas Data Analysis

This assignment demonstrates basic data analysis using the **Pandas library**.

## Concepts Covered
- Pandas Series creation
- Mathematical operations on Series
- Python functions on Series
- Creating and working with DataFrames
- Important DataFrame functions
- Data filtering and conditional selection
- GroupBy analysis
- Basic plotting using Pandas
- Simple sales data analysis

## File Structure

assignment10_pandas_analysis.py

## Tasks Implemented

### Task 1: Pandas Series Basics
- Create a Series from marks list
- Display values, index, and data type
- Access first element and last two elements

### Task 2: Mathematical Operations
- Add marks
- Subtract marks
- Multiply values
- Divide values

### Task 3: Series Analysis
- Maximum marks
- Minimum marks
- Total marks
- Average marks
- Check passed students using lambda

### Task 4: DataFrame Creation
- Create a DataFrame for students
- Display first rows and last rows
- Show shape and column names

### Task 5: DataFrame Functions
- info()
- describe()
- head()
- tail()
- Sort by marks
- Reset index

### Task 6: Filtering
- Students scoring above 75
- Students belonging to Math
- Students above average marks
- Students who failed

### Task 7: Grouping & Analysis
- Average marks per subject
- Student count per subject
- Maximum marks per subject

### Task 8: Pandas Plotting
- Bar chart for student marks
- Line chart of marks
- Histogram of marks

### Task 9: Sales Data Analysis
- Total revenue
- Average revenue
- Day with highest revenue
- Days where revenue > average
- Revenue vs Day graph
