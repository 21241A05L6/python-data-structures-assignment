# Assignment 11 – Matplotlib Visualization

This assignment demonstrates basic visualization techniques using Matplotlib.

## Concepts Covered

- Line Plot
- Scatter Plot
- Vertical and Horizontal Bar Charts
- Multiple Bar Charts
- Stacked Bar Charts
- Histogram
- Pie Chart

## File

matplotlib_assignment.py

Task 1 – Line plot showing sales trend  
Task 2 – Scatter plot for relationship between two variables  
Task 3 – Vertical and horizontal bar charts  
Task 4 – Multiple bar chart comparing yearly sales

Task 6 – Histogram showing numerical distribution  
Task 7 – Pie chart showing market share
